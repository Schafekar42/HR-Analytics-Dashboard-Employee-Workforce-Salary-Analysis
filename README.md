# HR-Analytics-Dashboard-Employee-Workforce-Salary-Analysis
HR Analytics Dashboard developed using SQL, Python, and Power BI to analyze workforce data. The project provides insights into employee headcount, salary distribution, recruitment sources, performance ratings, satisfaction levels, and gender diversity through interactive visualizations and KPI-driven reporting
